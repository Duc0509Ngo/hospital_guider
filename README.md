# Hospital guider

Phần mềm đăng ký khám chữa bệnh online theo từng khung giờ có trong ngày, điều này sẽ giúp giảm tải phải xếp hàng dài tại bệnh viện để đăng ký, tiêt kiệm nhiều thời gian cho bệnh nhân.
